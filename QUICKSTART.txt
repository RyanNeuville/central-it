INSTALLATION & RUNNING
═════════════════════════════════════════════════════════════════════════════

1. Install Dependencies:
   $ npm install

2. Run Development Server:
   $ npm run dev

3. Open Browser:
   http://localhost:3000

4. Build for Production:
   $ npm run build

5. Start Production Server:
   $ npm start

PROJECT STRUCTURE
═════════════════════════════════════════════════════════════════════════════

/app              → Pages & Routes (Home, Shop, Product, About, Support, Contact)
/components       → Reusable UI Components (Navbar, Footer, ProductCard, etc.)
/lib              → Data & Utilities (Products, Schemas, Helpers)
/public           → Static Assets
/styles           → Global CSS Configuration

CUSTOMIZATION
═════════════════════════════════════════════════════════════════════════════

Change Brand Name:
  • app/layout.tsx - Update title & description
  • components/layout/Navbar.tsx - Update logo text
  • components/layout/Footer.tsx - Update brand name

Update Products:
  • Edit lib/products.ts
  • Add/modify product objects

Customize Colors:
  • Edit tailwind.config.ts
  • Update color definitions in colors section

Add New Pages:
  • Create new folder in /app
  • Create page.tsx inside
  • Update navbar links

COMMON TASKS
═════════════════════════════════════════════════════════════════════════════

Add a New Product:
  1. Open lib/products.ts
  2. Add new product to array
  3. Increment product count

Change Colors:
  1. Edit tailwind.config.ts
  2. Update colors object
  3. Rebuild project

Create New Page:
  1. mkdir app/newpage
  2. Create page.tsx
  3. Add route to navbar

Troubleshooting
═════════════════════════════════════════════════════════════════════════════

TypeScript Errors:
  $ npm run typecheck

Build Fails:
  $ rm -rf .next node_modules
  $ npm install && npm run build

Port in Use:
  $ npm run dev -- -p 3001
